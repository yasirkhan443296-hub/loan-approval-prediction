# 📌 Project Overview

This project builds an end-to-end Credit Risk & Loan Approval Prediction System capable of assessing a borrower's financial profile and predicting whether a loan should be approved.

The project simulates a real-world banking and lending workflow by performing:

- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Feature Selection
- Data Preprocessing
- Class Imbalance Handling
- Model Training
- Hyperparameter Tuning
- Model Evaluation

The final solution helps financial institutions reduce lending risk and improve decision-making.

---

# 🎯 Business Problem

Financial institutions face significant losses due to loan defaults.

Traditional approval systems:

- Require manual review
- Are time-consuming
- Can be inconsistent
- May overlook hidden risk patterns

This project uses Machine Learning to:

✅ Predict loan approval

✅ Identify risky applicants

✅ Reduce default probability

✅ Improve lending efficiency

✅ Support data-driven decisions
